# snake_game
I've always been passionate about creating games, and this is my first game project. I developed the classic Snake Game, a game where you control a snake whose objective is to devour as much food as possible, but there's a catch: the more food you eat, the bigger the snake gets, making it increasingly difficult to control.
To play the snake game, you must download Python version 3.14.3 for Windows, Mac, or Linux. After that, you must install the latest pygame package in the terminal to run the code.
