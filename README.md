# snake_game
Sempre fui apaixonado por criar jogos, e este é meu primeiro projeto de jogo em python. Desenvolvi o clássico Jogo da Cobrinha, um jogo onde você controla uma cobra cujo objetivo é devorar o máximo de comidas possível, mas há um porém: quanto mais comida você come, maior a cobra fica, tornando cada vez mais difícil controlá-la.
Para rodar o jogo da cobrinha, você precisa baixar o Python versão 3.14.3 para Windows, Mac ou Linux. Depois disso, você precisa instalar o pacote pygame mais recente no terminal do seu IDE para executar o código e rodar o jogo.
